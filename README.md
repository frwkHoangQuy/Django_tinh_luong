## Cài Đặt

1. Đảm bảo Python và pip đã được cài đặt.
2. Clone repository này hoặc tải xuống dự án dưới dạng file zip và giải nén.
3. Mở terminal và di chuyển đến thư mục chứa dự án.
4. Cài đặt các phụ thuộc từ file requirements.txt bằng cách chạy lệnh sau:

    ```
    pip install -r requirements.txt
    ```

## Sử Dụng

1. Mở terminal và di chuyển đến thư mục gốc của dự án.
2. Chạy lệnh sau để khởi động máy chủ Django:

    ```
    py manage.py runserver
    ```

3. Mở trình duyệt và truy cập vào địa chỉ http://127.0.0.1:8000/ để truy cập ứng dụng.
